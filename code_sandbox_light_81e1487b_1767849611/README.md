# 性格タイプ診断ツール

## 📋 プロジェクト概要

このプロジェクトは、10個の質問を通してユーザーの性格を4つのタイプに分類する診断ツールです。直感的で使いやすいUIと、分析結果に基づいた詳細なフィードバックを提供します。

## 🎯 プロジェクトの目標

- **ユーザーエンゲージメント**: 楽しく参加できる診断体験の提供
- **自己理解促進**: 性格特性の理解とキャリアガイダンスの提供
- **使いやすさ**: レスポンシブデザインでモバイル・デスクトップ両対応
- **シェア機能**: 結果のソーシャルシェアによる拡散効果

## ✨ 主要機能

### 現在実装済みの機能

1. **診断システム**
   - 10個の多肢選択質問
   - 4つの性格タイプ分類（情熱型、思考型、協調型、創造型）
   - スコア計算とタイプ判定アルゴリズム

2. **ユーザーインターフェース**
   - モダンでレスポンシブなデザイン
   - 進行状況バー表示
   - スムーズなアニメーション効果
   - モバイルフレンドリーなレイアウト

3. **結果表示機能**
   - 詳細な性格タイプ説明
   - 個人の特徴リスト
   - キャリアアドバイス
   - スコア詳細表示
   - パーソナライズされたアドバイス

4. **インタラクション機能**
   - 前の質問に戻る機能
   - 自動進行オプション
   - 診断再実行機能
   - 結果シェア機能（Web Share API対応）

### 🎨 デザイン特徴

- **カラーテーマ**: 各性格タイプに対応したグラデーションカラー
- **アイコン**: Font Awesomeアイコンでビジュアル向上
- **タイポグラフィ**: Inter フォントファミリーを使用
- **レスポンシブ**: TailwindCSSによる完全レスポンシブデザイン

## 🎮 機能説明

### 4つの性格タイプ

1. **🔥 情熱型 (Passionate)**
   - 特徴: エネルギッシュで行動力抜群
   - 向いている職業: 営業、起業家、プロジェクトマネージャー

2. **🧠 思考型 (Analytical)**
   - 特徴: 論理的で分析力に優れる
   - 向いている職業: 研究者、エンジニア、コンサルタント

3. **❤️ 協調型 (Harmonious)**
   - 特徴: 思いやりがあり人を大切にする
   - 向いている職業: カウンセラー、教師、看護師

4. **💡 創造型 (Creative)**
   - 特徴: 独創的で自由な発想力を持つ
   - 向いている職業: デザイナー、アーティスト、ライター

### 診断アルゴリズム

各質問の選択肢は4つの性格タイプのいずれかに対応しており、ユーザーの回答に基づいてスコアを累積。最も高いスコアのタイプが結果として表示されます。

## 🚀 使用方法

1. **診断開始**: 「診断を開始する」ボタンをクリック
2. **質問回答**: 10個の質問に直感的に回答
3. **結果確認**: 性格タイプと詳細説明を確認
4. **シェア**: 結果をSNSやメッセージでシェア可能
5. **再実行**: 「もう一度診断する」で繰り返し利用可能
